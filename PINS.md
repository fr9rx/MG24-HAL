## Pin Mapping — XIAO MG24

| Arduino | Port | Pin | Alternate Function |
|---|---|---|---|
| D0  | C | 0 | A0 |
| D1  | C | 1 | A1 |
| D2  | C | 2 | A2 |
| D3  | C | 3 | A3 |
| D4  | C | 4 | A4 / SDA0 |
| D5  | C | 5 | A5 / SCL0 |
| D6  | C | 6 | A6 / TX0 |
| D7  | C | 7 | A7 / RX0 |
| D8  | A | 3 | A8 / SCK0 |
| D9  | A | 4 | A9 / MISO0 |
| D10 | A | 5 | A10 / MOSI0 |
| D11 | A | 9 | A11 |
| D12 | A | 8 | A12 |
| D13 | B | 2 | A13 / SCL1 |
| D14 | B | 3 | A14 / SDA1 |
| D15 | B | 0 | A15 / MOSI1 |
| D16 | B | 1 | A16 / MISO1 |
| D17 | A | 0 | A17 / SCK1 |
| D18 | D | 2 | A18 |
| LED | A | 7 | Onboard orange LED |

**Internal pins — never use as GPIO:**
- `PB04` — RF antenna switch
- `PB05` — RF switch power
- `PA06` — internal flash CS
- `PD03` — battery control
- `PD04` — battery ADC
